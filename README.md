# ☕ Java Fundamentals Study Project

🚀 Repositório de estudos práticos em Java focado em fundamentos da linguagem, especialmente manipulação de Strings, conversão de dados, formatação e classes utilitárias.

---

## 📚 Sobre o projeto

Este repositório contém exercícios desenvolvidos durante meus estudos de Java, com foco em:

- Manipulação de Strings
- Comparação de textos
- Busca e extração de informações
- Conversão de tipos
- Estruturação de dados
- Classes utilitárias do Java

O objetivo é consolidar a base da linguagem Java com prática constante e exemplos reais de uso.

---

## 🧠 Conteúdos estudados

### 🔤 Strings

- Criação de Strings
- Construtores de String
- Concatenação (`+`, `concat`)
- Extração de caracteres (`charAt`, `getChars`, `toCharArray`, `getBytes`)
- Comparação (`equals`, `equalsIgnoreCase`, `compareTo`)
- Busca (`indexOf`, `lastIndexOf`, `contains`)
- Manipulação (`substring`, `replace`, `trim`)
- Maiúsculas e minúsculas (`toUpperCase`, `toLowerCase`)
- Junção e separação (`join`, `split`)
- `StringBuilder` e `StringBuffer`
- `StringTokenizer`

---

### 📦 Parsing de dados

- Simulação de leitura de arquivos
- Separação de dados com delimitadores (`;` e `,`)
- Conversão de String para int (`Integer.parseInt`)
- Criação de objetos a partir de Strings

---

### 🧰 Classes utilitárias

- Introdução a classes utilitárias do Java
- Manipulação de arrays e bytes
- Base para futuros estudos em Data, Locale e Math
